## How to install and setup
1. **Prerequisites and Cloning the Repository**

Ensure that Git is installed on your computer, if it is not, download it here:

https://git-scm.com/install/

You should also ensure that Python 3 is installed on your computer, you can download it here: 

https://www.python.org/downloads/

Run this in the command terminal, this will clone the files from this repository:
```bash
git clone https://github.com/am-fine/PythonCalculatorProject
```
2. **Install Dependencies:**

The calculator requires the module PySide6 to run, to install it, navigate inside the project folder:
```bash
cd PythonCalculatorProject
```
Then run this line, this will install the necessary modules in `requirements.txt`
```bash
py -m pip install -r requirements.txt
```

## How to run
1. **Navigate to the project folder (if you are not in it already):**

Run this line from the same location you cloned the repository, this should put you in the project folder:
```bash
cd PythonCalculatorProject
```
2. **Execute the file**

Run this line, this will execute the file:
```bash
py SimpleCalculator[ADVANCED+].py
```
